# Question Count Feature - User Guide

## How to Use

### Creating a Quiz with a Target Question Count

1. Go to **Create Quiz** page
2. Fill in quiz details (title, description, etc.)
3. In the **"Number of questions"** section:
   - Click the **"Any number"** toggle to enable question count
   - The toggle will change to show a number
   - Use the **slider** or **number input** to set your target (1-60)

4. **Placeholder questions** will be automatically created
   - If you set 25 questions and have 5 existing, 20 empty slots are added
   - Each placeholder is ready for you to fill in
   - You can delete any slot if you change your mind

5. **Fill in your questions** as needed
   - The counter shows: "X/Y questions filled in"
   - You don't need to fill all slots before saving
   - Empty slots won't be saved in the final quiz

6. Click **"Create quiz"** to save

### Editing a Quiz's Question Count

1. Go to the quiz **Edit** page
2. Find the **"Number of questions"** section
3. To adjust:
   - **Increase count**: New empty questions are added at the end
   - **Decrease count**: Removes questions from the end (existing content is preserved)
   - **Disable**: Click toggle to allow unlimited questions

4. Click **"Save changes"**

### Rules

- ✅ Question count: 1 - 60 maximum
- ✅ Can be disabled for unlimited questions
- ✅ Empty question slots are automatically created
- ✅ Empty question slots won't be saved in the final quiz
- ✅ You can add/remove questions manually at any time
- ❌ Cannot set question count > 60
- ❌ Cannot set question count < 1 (when enabled)
